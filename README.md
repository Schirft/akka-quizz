# Akka Quiz

Webapp démo "Quiz of the Day" pour Akka, club d'investissement startup européen.

## Stack
- **Frontend:** Vite + React + Tailwind CSS
- **Backend:** Supabase (Auth, PostgreSQL, Edge Functions, RLS)
- **IA:** Claude Sonnet 4.5 (génération de questions)
- **News:** GNews API
- **Deploy:** Dokploy → app.hdconseils.com
